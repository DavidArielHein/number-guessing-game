# Number Guessing Game

A simple game of guessing a random number.

Project link: https://roadmap.sh/projects/number-guessing-game

---

## How to play

- **Welcome message**: First a welcome message will appear.
- **Select difficulty**: You have to select the difficulty (Easy, Medium or Hard)
- **Try to guess**: You will enter you guess, and get hints if your number is greater or lower than the answer.
